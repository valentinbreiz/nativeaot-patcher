High-level partition lifecycle (create / delete / resize / move) on top of Mbr and Gpt. Auto-detects the table type per call. MoveWithData(IBlockDevice, PartitionLocation, ulong) physically copies sectors before rewriting the table; the other operations are pure table edits.
