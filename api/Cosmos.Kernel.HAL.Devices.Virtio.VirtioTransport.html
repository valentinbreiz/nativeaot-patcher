Class VirtioTransport

Namespace
Cosmos.Kernel.HAL.Devices.Virtio
Assembly
Cosmos.Kernel.HAL.dll

Transport-independent core of a virtio device (virtio spec 1.x): device status handshake, feature negotiation, virtqueue setup, notifications, interrupts, and device-specific config access. Device drivers (VirtioNet, VirtioKeyboard, VirtioMouse) are written against this class only; VirtioMmioTransport and VirtioPciTransport implement the per-transport register access.
