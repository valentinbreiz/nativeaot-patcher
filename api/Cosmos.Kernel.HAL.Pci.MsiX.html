Programs the MSI-X capability of a PCI device (cap ID 0x11). Architecture-neutral: the message address/data are produced by the platform-registered MsiRouting backend, so the same table programming works whether interrupts ultimately land on an x64 LAPIC or an ARM64 GICv3 ITS.
