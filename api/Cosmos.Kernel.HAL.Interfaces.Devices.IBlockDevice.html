Interface IBlockDevice

Namespace
Cosmos.Kernel.HAL.Interfaces.Devices
Assembly
Cosmos.Kernel.HAL.Interfaces.dll

Interface for block storage devices (SATA, NVMe, virtio-blk, etc.). Block-level read/write only — partitioning and filesystems sit above this.

Error contract: ReadBlock(ulong, ulong, Span<byte>), WriteBlock(ulong, ulong, ReadOnlySpan<byte>) and Flush() throw on failure (device error, timeout, out-of-range request). A return without an exception means the operation completed on the device.
