Create(IBlockDevice)

Write a fresh, empty-but-signed MBR to LBA 0. No boot code, all four partition slots zeroed; callers add entries via WritePartition(IBlockDevice, int, byte, uint, uint). Also wipes LBA 1 (wipefs-style): Create claims the whole disk label, and a stale "EFI PART" header left there from a previous GPT layout would keep winning over this MBR in the partition scanner, which checks GPT first.
