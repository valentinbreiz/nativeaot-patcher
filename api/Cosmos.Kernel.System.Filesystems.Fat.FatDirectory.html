FindFreeRun(ReadOnlySpan<byte>, int, out bool)

Locate the first free entry slot (deleted or unused) where entriesNeeded consecutive 32-byte slots fit. Per spec the 0x00 terminator frees its own slot and everything after it, so scanning stops there and the remaining buffer counts as one free region — stale bytes past the terminator must never reset the run (or be treated as occupied). When the returned run overlaps the terminator, consumedTerminator is true and the caller must re-terminate after the new entries.

public static int FindFreeRun(ReadOnlySpan<byte> buffer, int entriesNeeded, out bool consumedTerminator)
