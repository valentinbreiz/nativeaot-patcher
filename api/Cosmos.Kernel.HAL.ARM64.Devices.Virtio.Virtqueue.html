Class Virtqueue

Namespace
Cosmos.Kernel.HAL.ARM64.Devices.Virtio
Assembly
Cosmos.Kernel.HAL.ARM64.dll

Virtqueue implementation for virtio devices. A virtqueue consists of three parts:

  • Descriptor table: array of buffer descriptors
  • Available ring: buffers offered to device
  • Used ring: buffers returned by device
