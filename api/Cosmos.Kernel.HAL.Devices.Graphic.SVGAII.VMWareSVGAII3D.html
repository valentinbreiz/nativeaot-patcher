Class VMWareSVGAII3D

Namespace
Cosmos.Kernel.HAL.Devices.Graphic.SVGAII
Assembly
Cosmos.Kernel.HAL.dll

SVGA3D command layer on top of SvgaIIDriver: surfaces, contexts, shaders, render state and DMA transfers, all submitted through the driver's command FIFO. Only meaningful when the device negotiated 3D support (Is3DEnabled) — QEMU's vmware-svga exposes no 3D capability, so this layer is only exercised on real VMware.
