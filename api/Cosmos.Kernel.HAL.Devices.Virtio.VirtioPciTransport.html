Class VirtioPciTransport

Namespace
Cosmos.Kernel.HAL.Devices.Virtio
Assembly
Cosmos.Kernel.HAL.dll

Virtio PCI transport (virtio spec section 4.1), modern interface only: the common/notify/ISR/device config regions are located through vendor-specific PCI capabilities pointing into BARs. Interrupts use MSI-X through the arch-neutral MsiX/MsiRouting path (LAPIC on x64, GICv3 ITS on ARM64); without MSI-X the device runs in polled mode. Legacy-only virtio-pci devices (I/O BAR interface, no vendor capabilities) are not supported.
