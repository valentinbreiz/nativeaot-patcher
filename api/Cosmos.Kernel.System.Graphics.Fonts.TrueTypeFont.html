Class TrueTypeFont

Namespace
Cosmos.Kernel.System.Graphics.Fonts
Assembly
Cosmos.Kernel.System.dll

Represents a TrueType font. Unlike the fixed-cell bitmap fonts, glyphs have individual widths and pair kerning, and the font can be rasterized at any pixel size. Data holds the raw bytes of the .ttf file; Width and Height are zero because a TrueType font has no fixed character cell. Rasterized glyphs are cached per (character, size) as grayscale coverage, so the same font instance can draw in any color without re-rasterizing.
