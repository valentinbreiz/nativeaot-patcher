Create(IBlockDevice)

Write a fresh GPT layout: protective MBR at LBA 0, primary header at LBA 1, and 32 zeroed sectors of 128-byte partition entries (LBAs 2..33). Backup header / array intentionally not written — callers targeting test images don't need them, and writing them needs CRC support that the boot path doesn't yet ship.
