Class SVGAII3DCanvas

Namespace
Cosmos.Kernel.System.Graphics
Assembly
Cosmos.Kernel.System.dll

Defines a VMWare SVGAII canvas implementation. Please note that this implementation of Canvas can only be used with virtualizers that implement SVGAII. This class will not work on regular hardware.
