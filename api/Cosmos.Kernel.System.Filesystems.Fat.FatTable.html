Class FatTable

Namespace
Cosmos.Kernel.System.Filesystems.Fat
Assembly
Cosmos.Kernel.System.dll

FAT12 / FAT16 / FAT32 table accessor and chain manager. All entry reads and writes go through the underlying IBlockDevice using FatBootSector geometry. Cluster numbers coming from on-disk metadata are untrusted: accessors treat anything outside the volume's data clusters as end-of-chain and never let it drive I/O.
