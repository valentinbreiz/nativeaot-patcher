Handle returned by Enable(PciDevice) identifying the mapped MSI-X table for a device. Carries the platform-binder's per-device state (e.g. ARM64 ITS DeviceID + ITT pointer) so subsequent SetEntry(MsiXContext, int, IrqDelegate, uint) calls can route through the same context.
