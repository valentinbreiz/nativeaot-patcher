TryFormat(ReadOnlySpan<char>, IVfsFormatOptions?)

Lay down a fresh on-disk filesystem on the backing store identified by source. Drivers cast options to their own concrete IVfsFormatOptions type; passing null selects driver defaults. The source must not be mounted: VfsManager.TryFormat/TryDestroy refuse a source with a live mount, since rewriting the volume underneath a superblock's cached geometry corrupts it.
